Tag Name Function
html Defines the text as an html element
head Contains basic unseen information about the page, such as the title
title The title is the name of the page, shown in the tab in the browser
body The second main part of the code after the head, which contains basically all the visible parts of the page
h1 h1, and the other h(number) tags, define a heading in the page, decreasing in size as the number gets bigger up to 6
p The p tag defines a paragraph
a Defines an html link, specified using an href attribute.
img Defines images, often paired with a src attribute, along with an alt, width and height attribute.
br This defines a line break
without any close to it. This is called an empty element.
b Makes text bold
strong Makes text strong, whatever that means
i Makes text italic
em Makes text emphasized, which for screen readers is different from italic
small c'mon guys, its average...
mark Marksthe text, aka highlights it, normally
del Gives the text generally a strikethrough, as this marks text that's been deleted
ins Opposite of del, this marks text that's been inserted, usually underliningit.
sub and sup subscript and superscript text
pre Defines preformatted text, preserving spaces and line breaks
table Makes a table
tr Defines each table row
th Defines the table headers
td Table data, the content of each cell
caption Defines a table caption
colgroup Specifies a croup of one or more columns for formatting
col Specifies column properties for columns in a colgroup element
thead Groups the header content in a table
tbody Groups the body content
tfoot Footer content
